## 🛠️ Troubleshooting

If you do not see the content, check these common points.

### The files are in the wrong place
The files must reside in the same folder as the game executable. If you put them in a folder deeper than the main directory, the game will ignore them. Move them to the root folder.

### Steam overwrote the files
If you verify your game files through Steam, Steam will delete these modifications. You must re-extract the files into the folder every time you verify game integrity.

### Missing administrative rights
Some versions of Windows restrict file changes in the Program Files directory. If the game does not seem to change, try running Steam as an administrator. Right-click the Steam icon and choose Run as administrator.
